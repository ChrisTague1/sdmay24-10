## Deployment

> Work In Progress

Deployments happen automatically on pushes to main, via `vercel`

## Running

If you do not have `node` installed, [install it](https://nodejs.org/en)

```bash
# to see if you have node
node --version
```

To install dependencies:

```bash
npm ci
```

To test locally, run:

```bash
npm start
```
